# useIsBrowser()

Detect if code is running where browser features are available.

## Usage

```tsx copy
import { useIsBrowser } from '@norr/hooks';

const Component = () => {
  const isBrowser = useIsBrowser();

  if (isBrowser) {
    return (
      <span>
        Browser features are available.
      </span>
    );
  }

  return null;
};
```
